NaviSafe

En moderne rapporteringsløsning for luftlige hindringer, utviklet med fokus på sikker navigasjon og enkel deling av viktig informasjon for piloter og luftfartsmyndigheter.

💻

Hovedidé

NaviSafe gir pilotsikkerhet ved å samle rapporter om luftlige hindringer i ett intuitivt system, og gjør det enkelt å finne oppdatert informasjon før flygning. Mer informasjon finner du her: GitHub

Tilbake til prosjekter Kontakt meg

Hva prosjektet inneholder

  • Rapporteringsskjema for hindringer
  • Datavisning med søk og filtre
  • Rollebasert visning for pilot og admin
  • Responsivt og tydelig grensesnitt

Tech Stack

Frontend: HTML, CSS, Bootstrap CSS, Leaflet, JavaScript

Backend: ASP.NET Core 9.0, MVC + Razor, EF Core, OpenTelemetry, JWT

Database: MariaDB 11.8, phpMyAdmin

Infrastruktur: .NET Aspire, Docker

Mapping: OpenStreetMap, Leaflet Draw

Hva jeg lærte

Jeg lærte å kombinere sikkerhet og brukervennlighet, og hvordan bygge et pålitelig system for kritisk luftfartsinformasjon.
